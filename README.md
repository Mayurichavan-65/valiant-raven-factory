# 🍱 Food Redistribution System

A modern web platform that helps reduce food waste by connecting food donors with NGOs, volunteers, and needy communities.

## 🚀 Features

* Multi-role authentication
* Food donation management
* NGO & volunteer dashboards
* Real-time donation tracking
* Smart matching system
* Responsive modern UI

## 🛠️ Tech Stack

* Next.js
* React.js
* Tailwind CSS
* Supabase
* TypeScript

## 🌍 Mission

Turning surplus food into support for communities in need.

## ⚙️ Run Locally

```bash id="9y7r8d"
npm install
npm run dev
```
